# 라프라시아 야구 시뮬레이션 RPG
## "Diamond & Magic" - 게임 문서

---

## 🎮 게임 개요

**장르**: 텍스트 기반 야구 시뮬레이션 RPG  
**플랫폼**: Python (콘솔 기반)  
**대상 연령**: 전체 이용가  
**플레이 시간**: 1 회차 약 2-3 시간 (멀티 엔딩)

---

## 📖 스토리

### 배경
라프라시아 780 년, 마법과 과학이 공존하는 대륙에서 야구는 단순한 스포츠를 넘어 예술로 취급받습니다.

### 주인공
**카일 윈저** (8 세, 휴먼)
- 하플랜드 로얄 야구 아카데미 코치과 입학생
- 전생에서 현대 야구 지식을 가지고 전이함
- 목표: 프로팀 전략 코치

### 주요 등장인물
| 이름 | 종족 | 역할 | 특징 |
|------|------|------|------|
| 에릴 | 엘프 | 투수 | 정교한 커브, 프로 스카우트 대상 |
| 토비 | 하플링 | 데이터 분석가 | 세이버메트릭스 전문가 |
| 레이몬드 | 휴먼 | 스피드스터즈 감독 | 카일의 인턴십 멘토 |
| 그리폰 | 휴먼 | 아카데미 주임강사 | 엄격하지만 정의로운 성격 |

## 🎯 게임 시스템

### 1. 상태 시스템
```
[플레이어 스탯]
├─ STR (힘): 1-20
├─ DEX (민첩): 1-20
├─ INT (지성): 1-20
├─ CHA (매력): 1-20
└─ Baseball Knowledge: 0-200

[컨디션]
├─ 체력 (Stamina): 0-100
└─ 기분 (Mood): 0-200
```

### 2. 관계 시스템
```
[NPC 호감도]
├─ 0-20: 적대
├─ 21-40: 낯섦
├─ 41-60: 보통
├─ 61-80: 우호
├─ 81-100: 친구
└─ 100+: 특별한 관계
```

### 3. 시간 시스템
- 게임 내 날짜: 라프라시아 780 년 3 월 17 일 시작
- 시간 경과: 선택에 따라 1-4 시간 소모
- 예정된 이벤트: 특정 날짜에 자동 발생

### 4. 인벤토리 시스템
- 아이템 획득/사용/폐기
- 주요 아이템: 전략 노트, 훈련 장비, 선물 등

## 🎮 플레이 방법

### 기본 조작
```
1. 상태 화면 확인
2. 현재 상황 읽기
3. 선택지 번호 입력 (1-4)
4. 결과 확인
5. 반복
```

### 명령어
- `1-9`: 선택지 선택
- `q`: 게임 저장 및 종료
- `s`: 상태 상세 확인
- `i`: 인벤토리 확인
- `l`: 로그 확인

## 🔧 확장 방법

### 새 장면 추가
1. `scenes.json` 에 새 장면 데이터 추가
2. `engine.py` 의 `load_scenes()` 메서드 수정
3. 선택지의 `next_scene` 필드에 새 장면 ID 연결

### 새 NPC 추가
1. `world_state.json` 의 `relationships` 에 추가
2. `npcs.json` 에 상세 데이터 작성
3. 호감도 이벤트 작성

### 새 이벤트 추가
1. `events.json` 에 이벤트 데이터 추가
2. 특정 날짜/조건에서 트리거 설정
3. 이벤트 완료 보상 작성
