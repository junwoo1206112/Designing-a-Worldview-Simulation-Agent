### Windows
```
1. game\start.bat 더블클릭
```

## 💡 게임 팁

### 1. 체력 관리
- 체력이 0 이 되면 강제 휴식
- `휴식` 선택지로 체력 회복
- 중요한 이벤트 전에 체력 확보

### 2. 관계 관리
- 호감도 80 이상: 친구解锁
- 호감도 100: 특별한 이벤트 발생
- 각 NPC 마다 선호하는 행동이 다름

| NPC | 선호 행동 |
|-----|----------|
| 에릴 | 투구 분석, 훈련 코칭 |
| 토비 | 세이버메트릭스 연구 |
| 레이몬드 | 업무 보고, 성실함 |

### 3. 스탯 성장
- `baseball_knowledge` 가 핵심 스탯
- 150 이상: 인턴십 성과 좋음
- 200: 모든 지식 관련 선택지解锁

### 4. 시간 관리
- 각 행동은 1-4 시간 소모
- 예정된 이벤트 날짜 확인
- 데드라인 전에 준비 완료

## 🐛 문제 해결

### 게임이 실행되지 않음
```
1. Python 3.7+ 설치 확인
2. 인코딩 오류: start.bat 사용 (Windows)
3. 파일 권한 확인 (Linux/Mac)
```

### 세이브 파일이 없음
```
- game/saves/ 폴더 확인
- 처음 실행 시 자동 생성됨
```

### 한글이 깨짐
```
- 터미널 인코딩을 UTF-8 로 설정
- Windows: chcp 65001 실행
```
